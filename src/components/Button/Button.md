### Variant:

```js
<Button>Click Me</Button>

<Button variant='dark' style={{marginLeft: '5%'}}>Click Me</Button>

<Button variant='dark' semantic='danger' style={{marginLeft: '5%'}}>Click Me</Button>
```

### Semantic:

```js
<Button semantic='danger'>Click Me</Button>
<Button semantic='success' style={{marginLeft: '5%'}}>Click Me</Button>
<Button style={{marginLeft: '5%'}}>Click Me</Button>
```

Sizing:

```js
<Button width='500px'>Click Me</Button>
<Button semantic='success' style={{marginLeft: '5%'}} width='100px'>Click Me</Button>
<Button style={{marginLeft: '5%'}} height='100px'>Click Me</Button>
```
